# Data and reproduction

## v2 correction note (read first)
Every ISJ/Botev bandwidth through v1 was computed with KDEpy's `improved_sheather_jones`, which
undersmooths ~2.6x versus the Botev-Grotowski-Kroese fixed point. v2 embeds the direct fixed
point (`scripts/ad_kde_v31.py`; `scripts/ad_kde_v30.py` is a compatibility shim; KDEpy dropped).
No table in this paper prints an ISJ number (the exact-sample comparison is qualitative prose,
re-verified against the corrected implementation), so no values are restated; the correction is
for script integrity and the prose claim's provenance.

## Mixed-grain heaping / subgroup-lattice peel (Table: mixed)
`scripts/exp_nhanes_heaping_v16.py`: the mixed-grain NHANES companion study (run locally with the
same --data-dir as v15); imposes grain mixtures (5, 10, 20 lb) with a dominant fine grain on the
measured weights and prints recovered grains, weights, and the exact-report share against the
planted values, alongside residue / forced single-pitch / lattice ISE.
`scripts/exp_heaping_mixed_v1.py`: grain mixtures over {0.125, 0.25, 0.5} (three weight settings,
one with a planted unrounded share) plus a non-dyadic nesting {0.15, 0.45}, a rare-coarse
adversarial mixture, single-grain reduction, and exact-abstention checks; Gaussian and strongly
skewed targets, n=8000, reps=15, seed0=20260627. Writes `results/exp_heaping_mixed_v1.json`; the
paper's tab:mixed prints the Gaussian rows. Fully synthetic, no external data.

## NHANES with the first-order peel (pending local run)
`scripts/exp_nhanes_heaping_v15.py`: v14's controlled Part 1 plus the AD-1 peel arm (padded
analysis grid, grid-native fallback on abstention). Run from scripts/ with the NHANES 2017-2018
files (DEMO_J.xpt, BMX_J.xpt, SMQ_J.xpt) in --data-dir; prints the Part 1 table with the peel
column and detected pitch, regenerates both figures. v14 retained unchanged for provenance.

## Coarse heaping / first-order peel (Table: coarse)
GATE VERSION: all results (the synthetic battery in results/ and the NHANES table
tab:heaping_real) are under the v2.3 gate (quietest-stretch detection valley, unbiased gap-median
floor, n-free core-to-floor capacity at 5.0 nats, z=5 primary / 3.5 harmonic confirmation). The
NHANES v15 rerun under this gate reconciled the peel and pitch columns: D=5..30 unchanged to the
reported precision; D=40 now capacity-abstains with the fallback reproducing the residue column
exactly (the prior gate peeled there for 0.016; the conservatism is the documented price of the
guard).
`scripts/exp_heaping_coarse_v1.py`: three targets (Gaussian, strongly skewed, claw), pitches
Delta in {0.1, 0.25, 0.5} plus exact, n in {2000, 8000}, reps=15, seed0=20260627
(rng = 20260627 + 7919*r + n), M=8192 over [-8, 8], ISE against the true pre-rounding density.
Writes `results/exp_heaping_coarse_v1.json`; the paper's table prints the n=8000 rows. Exact
regimes verify the 100% abstention invariant. Fully reproducible with no external data.

## Synthetic heaping (Table I, Fig. 1)
Generated by `scripts/ad_kde_v31.py` (the method module also runs the heaped-data robustness
experiment). Marron-Wand test densities, samples rounded to 0.1, n=2000, deterministic seeds set in
the script. Running `python3 scripts/ad_kde_v31.py` reproduces `figures/fig_kde_heaped.pdf` and prints
the Table I values. Fully reproducible with no external data.

## NHANES self-reports (Table II, Figs. 2-3)
National Health and Nutrition Examination Survey 2017-2018, public files from the U.S. CDC
(https://www.cdc.gov/nchs/nhanes/). Two variables:
- measured adult weight from the body-measures file (BMXWT); n=5173 after the standard adult screen;
- self-reported cigarettes per day from the smoking questionnaire (SMD650); 1019 adult smokers
  reporting a daily count.
The data are public but not redistributed here. Download the 2017-2018 files and run
    python3 scripts/exp_nhanes_heaping_v14.py --data-dir <NHANES_DIR>
to regenerate `figures/fig_heaping_weight.pdf`, `figures/fig_heaping_cigarettes.pdf`, and the Table II
values. Requires `pandas`.

## The actual cigarette counts (natural-heaping paragraph, one sentence)
`scripts/exp_cig_lattice_probe_v1.py` (author-run, n=1019, single dataset, seed-free): unit-pitch
anchored lattice reader with beyond-extent amplitude reads and instrument-unit candidates; result
of record: grains {20,10,5,1}, weights (0.161, 0.120, 0.183, 0.405) + residual 0.131 (integer
total 0.536); observed m5/m10/m20 = 0.570/0.425/0.216 vs forward-predicted 0.471/0.322/0.189
(systematic deficit matching ground-truthed surrogates whose weights are accurate to a few
hundredths). Consistency demonstration, no ground truth; the manuscript cites the decomposition and
the tracking, not an error number.
